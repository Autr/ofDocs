ofBoxPrimitive

editEdit

The ofBoxPrimitive allows you to create cubes and cuboids.

Like other primitives, it allows you to set its dimensions and position, draw it, etc. As a simple example:


void setup()
{

	box.set( boxSize );
	// or
	// box.set( boxHeight, boxWidth, boxDepth );
}
